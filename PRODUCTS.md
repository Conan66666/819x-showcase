# 产品矩阵

819X 框架以**两大产品线 / 三套底座**承接绝大多数白标交易所需求。

---

## 第一块 · 纯合约交易所

适合**主打衍生品交易**、运营侧需要**代理体系/分润**的品牌方。

### 主力底座

**代理体系完备版**

- 多级代理 + 分润链路
- 标准撮合 / 行情 / K 线 / 充提 / KYC
- Filament 运营后台
- 客户端：响应式 PC + 移动端
- 起站周期：1-3 天换皮

### 风控变形

**冻结/解冻资金版**

在主力底座基础上增加：
- 用户资金状态管理（冻结 / 部分冻结 / 解冻）
- 风控钩子：与提现、交易状态联动
- 运营后台资金明细追踪
- 适合需要**精细化资金管控**的业务形态

---

## 第二块 · 合约 + 期货交易所

适合面向**多产品线交易**、运营侧需要**行情干预与精细化合约控制**的品牌方。

### 主力底座

**拉盘 + 单控版**

- 合约 + 期货双产品线
- 拉盘能力：运营侧行情可干预
- 单控合约：运营侧风险/收益精细化控制
- 集成客服系统（独立子域 + 工单 + 实时对话）
- Filament 多面板：管理 / 客服 / 代理三栏后台
- 起站周期：1-3 天换皮（含客服系统）

### 配套能力

- 行情中继接入（高并发 fanout）
- 多语种 i18n
- 多币种 / 多市场配置

---

## 选型指南

**选第一块·主力底座** 如果：
- 主打衍生品 / 合约
- 需要代理分润体系
- 业务模式标准化

**选第一块·风控变形** 如果：
- 业务有特殊资金管控诉求
- 需要冻结/解冻流程

**选第二块** 如果：
- 多产品线（合约+期货）
- 需要拉盘 / 单控
- 需要客服系统
