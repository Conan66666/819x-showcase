# 产品矩阵

819X 白标交易所框架以**两大产品线 / 三套底座**承接绝大多数白标交易所需求。
可订阅交易对涵盖：外汇 (Forex)、虚拟币 (Crypto)、贵金属 (Metals)、原油 (Crude Oil)、美股 (US Stocks)、港股 (HK Stocks)等领域的高频数据。

---

## 第一块 · 纯合约交易所（二元期权秒合约）

适合**主打衍生品交易**、运营侧需要**代理体系/分润**的品牌方。

### 主力底座

**代理体系完备版**

- 多级代理 + 分润链路
- 标准撮合 / 行情 / K 线 / 充提 / KYC / 控盘
- 单控合约：运营侧风险/收益精细化控制
- 集成客服系统（独立子域 + 工单 + 实时对话）
- 客户端：响应式PC端 + 移动端H5 + APP + 桌面软件
- 专业易上手的运营后台：管理 / 客服 / 代理三栏后台
- 起站周期：1-3 天

### 风控变形

**冻结/解冻资金版**

在主力底座基础上增加：
- 用户资金状态管理（冻结 / 部分冻结 / 解冻）
- 风控钩子：与提现、交易状态联动
- 运营后台资金明细追踪
- 适合需要**精细化资金管控**的业务形态

### 配套能力

- 行情中继接入（高并发 fanout）
- 多语种 i18n
- 多币种 / 多市场配置
