# 能力栈

819X白标交易所框架不只是「一份代码」 —— 而是**底座 + 工程能力 + 运维 SOP + 经验库**的综合产品。

---

## 一、快速起站工程

### 1-3 天起新站，能做到什么程度？

**视觉资产层**
- Logo / 主题色 / 站点名 / SEO meta 全替换
- 多语种文案统一注入
- 落地页 / 营销页可选附带
- 智能客服系统
- 专业易上手的运营后台

**域名 + SSL**
- 主站 + 后台 + 客服三子域统一配置
- Cloudflare 橙云接入（含跨子域 SSL 跳坑经验）
- SSL 自动续签

**自检与上线**
- 部署后多点自检（资产 / WS / DB / Redis / FPM）
- 真实浏览器 E2E 验证关键路径

## 二、可定制深度

按客户需求分级，从「换皮即用」到「业务深度改造」均覆盖：

**L0 · 换皮上线** 1-3 天
- 仅视觉更换 + 文案 + 域名

**L1 · 配置定制** 1-2 周
- 撮合参数 / 风控规则 / 手续费策略 / 代理分润比例
- 后台菜单与权限调整
- 邮件 / 短信 / 推送模板 / Telegram机器人 / Whatsapp推送 / LINE推送

**L2 · 业务定制** 按需排期
- 新增业务模块（如特殊产品形态）
- 接入指定第三方（行情源 / 支付通道 / KYC 服务商）
- 独立 API 对外开放

**L3 · 深度共创** 按需排期
- 全新撮合策略
- 新增产品线（合约外扩展）
- 多租户私有化部署
